RTF generator.

Package Details

Converts chemical shift table from NMR-STAR loop to RTF table.

Classes

{@link EDU.bmrb.rtfgen.rtf rtf} contains constants: RTF codes, amino acid labels, etc.

{@link EDU.bmrb.rtfgen rtfgen} is the executable class that handles the conversion.

{@link EDU.bmrb.rtfgen.shiftdata shiftdata}: data structure that holds chemical shifts table.

{@link EDU.bmrb.rtfgen.pair pair}, {@link EDU.bmrb.rtfgen.triple triple}, and {@link EDU.bmrb.rtfgen.VectorSorter VectorSoter}: utility classes for sorting the table etc.

Usage

java EDU.bmrb.rtfgen.rtfgen input file sort order output file

  • input file: NMR-STAR file with chemical shifts. It can be just the chemical shifts loop: the program will attempt to add data block and saveframe names required by STARlib.
  • sort order: 1 -- by chemical shift number, 2 -- by amino acid.
  • output file: use dash '-' to print to stdout.
