RTFGen: NMR-STAR 3.1 to RTF table converter

Input file type:
Complete BMRB entry
Chemical shifts loop only

Output options:
Sort table by residue sequence number
Sort table by residue code

Input file types

1. Complete BMRB entry can contain multiple assigned chemical shifts saveframes and multiple entities (polymers) in the assembly (molecular system). RTFGen will create separate tables for each saveframe and/or entity assembly ID, using assembly name, entity assembly name, and sample conditions for the titles.

2. When uploading chemical shifts loop only, RTFGen will generate a single table "Table 1". Input file must contain data for single entity only and must have at least the following columns and values: 

    loop_
        _Atom_chem_shift.Comp_index_ID 
        _Atom_chem_shift.Comp_ID 
        _Atom_chem_shift.Atom_ID 
        _Atom_chem_shift.Val 
        
          ... ... ... ... 
          ...
          
    stop_
